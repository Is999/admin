# 任务系统

本目录集中归档 `admin` 任务系统相关文档，只保留框架级说明：运行模式、队列、Worker、Scheduler、工作流和任务查询。

## 文档分工

- [任务系统使用手册](任务系统使用手册.md)
  - 面向运维、测试、后端
  - 说明 `run_mode`、控制面/执行面拆分、队列职责、共享 Redis 与 `app_id` 隔离
- [任务系统接口总览](接口文档/任务系统/任务系统接口.md)
  - 面向前端、测试、联调
  - 按后台菜单查看任务总控、任务队列、任务列表、任务监控接口
- [任务队列与工作流指南](角色文档/后端开发/任务队列与工作流指南.md)
  - 面向后端开发
  - 聚焦插件化接入、任务注册、CLI、开发扩展约束

## 推荐查看顺序

1. 日常使用、部署或验收时，先看 [任务系统使用手册](任务系统使用手册.md)。
2. 需要联调后台页面或接口时，再看 [任务系统接口总览](接口文档/任务系统/任务系统接口.md)。
3. 需要新增任务、工作流或周期调度时，再看 [任务队列与工作流指南](角色文档/后端开发/任务队列与工作流指南.md)。
4. 任务未执行、周期未触发、Redis 异常时，回到 [任务系统使用手册](任务系统使用手册.md) 的页面观察顺序和周期任务检查项。
