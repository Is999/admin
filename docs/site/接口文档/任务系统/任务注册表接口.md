# 任务注册表接口

---

## 1. 查询任务类型注册表

### 请求示例

```http
GET /api/tasks/registry/task-types
Accept-Language: zh-CN
```

### 响应示例

```json
{
  "status": true,
  "code": 1,
  "message": "查询成功",
  "data": {
    "items": [
      {
        "taskType": "workflow:trigger",
        "description": "工作流触发入口任务，由总控台触发工作流时自动入队",
        "usageHint": "通常不建议直接手动投递该类型；优先使用“手动触发工作流”表单。",
        "payloadExample": "{\n  \"workflowId\": \"wf-demo-001\"\n}",
        "manualRecommended": false
      }
    ]
  }
}
```

### 响应字段说明

| 字段 | 类型 | 是否必须 | 默认值 | 字段解释 |
|------|------|----------|--------|----------|
| status | bool | 是 | false | 状态: true 成功, false 失败 |
| code | int | 是 | 1 | 业务状态码 |
| message | string | 否 |  | 消息 |
| data | object | 是 |  | 任务类型注册表 |

#### data 内部字段

| 字段 | 类型 | 是否必须 | 默认值 | 字段解释 |
|------|------|----------|--------|----------|
| items | array | 是 | [] | 任务类型列表 |

#### items[] 内部字段

| 字段 | 类型 | 是否必须 | 默认值 | 字段解释 |
|------|------|----------|--------|----------|
| taskType | string | 是 |  | 任务类型 |
| description | string | 否 |  | 按 `Accept-Language` 返回的说明 |
| usageHint | string | 否 |  | 按 `Accept-Language` 返回的使用提示 |
| payloadExample | string | 否 |  | 推荐人工投递 payload 示例 |
| manualRecommended | bool | 是 | false | 是否推荐人工手动投递 |

---

## 2. 查询工作流注册表

### 请求示例

```http
GET /api/tasks/registry/workflows
Accept-Language: zh-CN
```

### 响应示例

```json
{
  "status": true,
  "code": 1,
  "message": "查询成功",
  "data": {
    "items": [
      {
        "name": "cache.refresh",
        "description": "批量刷新共享缓存工作流",
        "defaultQueue": "maintenance",
        "nodeCount": 2,
        "usageHint": "适合按目标列表触发缓存刷新；targets 可填写缓存目标或页面带入的缓存键。",
        "targetsExample": "admin_profile:1, admin_roles_detail:1"
      }
    ]
  }
}
```

### 响应字段说明

| 字段 | 类型 | 是否必须 | 默认值 | 字段解释 |
|------|------|----------|--------|----------|
| status | bool | 是 | false | 状态: true 成功, false 失败 |
| code | int | 是 | 1 | 业务状态码 |
| message | string | 否 |  | 消息 |
| data | object | 是 |  | 工作流注册表 |

#### data 内部字段

| 字段 | 类型 | 是否必须 | 默认值 | 字段解释 |
|------|------|----------|--------|----------|
| items | array | 是 | [] | 工作流列表 |

#### items[] 内部字段

| 字段 | 类型 | 是否必须 | 默认值 | 字段解释 |
|------|------|----------|--------|----------|
| name | string | 是 |  | 工作流名称 |
| description | string | 否 |  | 按 `Accept-Language` 返回的说明 |
| defaultQueue | string | 是 |  | 默认执行队列 |
| nodeCount | int | 是 | 0 | 工作流节点数量 |
| usageHint | string | 否 |  | 按 `Accept-Language` 返回的使用提示 |
| targetsExample | string | 否 |  | 执行目标填写示例 |
