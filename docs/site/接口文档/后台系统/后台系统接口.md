# 后台系统接口总览

为了便于查阅和维护，admin-cron 后台系统的管理接口文档已根据功能模块（对应管理后台菜单）进行了拆分。

请查阅以下具体功能模块的接口文档：

- [基础规范与认证接口](接口文档/后台系统/基础规范与认证接口.md)
  - 接口通用返回结构、鉴权规范（Token）
  - 登录、权限码、刷新与退出机制
  - 接口签名验签与请求体加密规则
- [接口文档访问接口](接口文档/后台系统/接口文档访问接口.md)
  - 为文档站创建受限 HttpOnly 会话
  - 文档静态资源复用 `docs.index` 权限校验
- [个人信息接口](接口文档/后台系统/个人信息接口.md)
  - 管理员个人信息获取与修改
  - 修改密码、MFA 状态及头像更新
- [管理员管理接口](接口文档/后台系统/管理员管理接口.md)
  - 用户列表、详情、新增与编辑
  - 用户状态控制、重置密码与角色绑定
- [角色管理接口](接口文档/后台系统/角色管理接口.md)
  - 角色列表、角色树查询与增删改查
  - 角色权限绑定
- [权限管理接口](接口文档/后台系统/权限管理接口.md)
  - 权限列表、权限树查询与增删改查
- [系统配置接口](接口文档/后台系统/系统配置接口.md)
  - 系统常量配置的列表查询与维护
  - 配置缓存管理
- [缓存管理接口](接口文档/后台系统/缓存管理接口.md)
  - 内置缓存目标、Redis 服务信息查询
  - Redis Key 搜索与详情查询、手动刷新缓存
- [日志管理接口](接口文档/后台系统/日志管理接口.md)
  - 后台管理员操作日志查询
- [消息中心接口](接口文档/后台系统/消息中心接口.md)
  - 后台管理员站内信与通知（收件箱、未读数量、铃铛通知、发送与已读/删除）
- [秘钥管理接口](接口文档/后台系统/秘钥管理接口.md)
  - 前端应用秘钥列表、状态管理及缓存刷新
- [文件传输接口](接口文档/后台系统/文件传输接口.md)
  - 分片上传、状态查询、完成上传、文件下载与访问
- [安全调试接口](接口文档/后台系统/安全调试接口.md)
  - 签名、验签、加密、解密联调工具

## 统一响应字段说明

| 字段 | 类型 | 是否必须 | 默认值 | 字段解释 |
|------|------|----------|--------|----------|
| status | bool | 是 | false | 状态: true 成功, false 失败 |
| code | int | 是 | 1 | 业务状态码 |
| message | string | 否 |  | 消息 |
| data | object/array/null | 否 |  | 后台系统接口响应数据 |
