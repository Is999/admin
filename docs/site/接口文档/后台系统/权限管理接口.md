# 权限管理接口

---

## 1. 查询权限列表

### 请求示例

```http
GET /api/permissions?page=1&pageSize=10&module=system&status=1
```

### 查询参数

| 字段 | 类型 | 是否必须 | 默认值 | 字段解释 |
|------|------|----------|--------|----------|
| page | int | 否 | 1 | 页码 |
| pageSize | int | 否 | 10 | 每页数量 |
| module | string | 否 |  | 权限模块 |
| title | string | 否 |  | 权限名称 |
| uuid | string | 否 |  | 权限码 |
| status | int | 否 |  | 状态，1 启用，0 禁用 |

### 响应示例

```json
{
  "status": true,
  "code": 1,
  "message": "查询成功",
  "data": {
    "list": [
      {
        "id": 1,
        "uuid": "admin.list",
        "title": "查询管理员列表",
        "module": "system",
        "pid": 0,
        "type": 2,
        "status": 1
      }
    ],
    "total": 1
  }
}
```

### 响应字段说明

| 字段 | 类型 | 是否必须 | 默认值 | 字段解释 |
|------|------|----------|--------|----------|
| status | bool | 是 | false | 状态: true 成功, false 失败 |
| code | int | 是 | 1 | 业务状态码 |
| message | string | 否 |  | 消息 |
| data | object | 是 |  | 权限分页数据 |

#### data 内部字段

| 字段 | 类型 | 是否必须 | 默认值 | 字段解释 |
|------|------|----------|--------|----------|
| list | array | 是 | [] | 权限列表 |
| total | int | 是 | 0 | 总数 |

## 7. 删除权限

### 请求示例

```http
DELETE /api/permissions/1001
```

### 路径参数

| 字段 | 类型 | 是否必须 | 默认值 | 字段解释 |
|------|------|----------|--------|----------|
| id | int | 是 |  | 权限 ID |

### 响应示例

```json
{
  "status": true,
  "code": 1,
  "message": "删除成功"
}
```

### 响应字段说明

| 字段 | 类型 | 是否必须 | 默认值 | 字段解释 |
|------|------|----------|--------|----------|
| status | bool | 是 | false | 状态: true 成功, false 失败 |
| code | int | 是 | 1 | 业务状态码 |
| message | string | 否 |  | 消息 |

### 高危操作说明

- 删除权限会级联删除全部下级权限。
- 删除后会清理角色权限关系并刷新权限缓存，使权限树和权限码映射立即生效。
